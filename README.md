# scholarship-dashboard

# Installation and Setup: 
1. Clone the repository: `git clone <repository-url>`
2. Install dependencies: `npm install`
3. Run the application: `npm run dev`

# Experience with Claude
Working with Claude Code was a highly productive experience that allowed me to bypass the 'blank page' phase of a new framework and focus on core logic. While it was an excellent resource for explaining Nuxt 3 conventions, it required careful human oversight to catch small errors.
